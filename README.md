# 🚗 Gesture-Controlled IoT Robotic Car using ESP8266

## 📌 Project Overview
This project presents a Gesture-Controlled Robotic Car using IoT technology. It uses two ESP8266 microcontrollers to enable wireless communication between a gesture transmitter and a robot car. The system allows users to control the car using simple hand movements.



## 🎯 Features
- Real-time gesture-based control
- Wireless communication (ESP-to-ESP)
- Low-cost and easy to build
- No need for complex modules like NRF24
- Beginner-friendly IoT project



## 🛠️ Components Used
- ESP8266 (NodeMCU) × 2
- MPU6050 Accelerometer/Gyroscope
- L298N Motor Driver
- DC Motors & Wheels
- Robot Chassis
- Battery Pack
- Jumper Wires



## ⚙️ Working Principle
- The transmitter ESP8266 reads hand gestures using the MPU6050 sensor.
- The gesture data is processed and sent wirelessly to the receiver ESP8266.
- The receiver controls the motors based on the received gesture:
  - Forward
  - Backward
  - Left
  - Right

## 🚀 How to Run
1. Install Arduino IDE
2. Add ESP8266 board in Board Manager
3. Upload transmitter code to one ESP8266
4. Upload receiver code to another ESP8266
5. Connect hardware as per circuit diagram
6. Power ON and control using hand gestures
